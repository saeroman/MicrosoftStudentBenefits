### 아래는 전세계 모든 학생을 대상으로 Microsoft에서 제공하는 Student Benefits에 대한 안내입니다.

## Microsoft Azure for Students
🔹대학생(Full‑time) 대상 신규 사용자에게 매년 $100 크레딧 + 12개월 무료 20여개 인기 서비스, 이후 65개 이상 영구 무료 서비스 제공
🔹신용카드 불필요

### Azure for Students Starter
중·고등학생 대상 주요 서비스 일부(웹앱, 함수, DevOps, MySQL 등)에 대한 영구 무료 접근 <br>

🧭 **신청 절차**
1. [Azure for Students 페이지] 방문 https://azure.microsoft.com/ko-kr/free/students/
2. “Start free” 또는 “Activate” 클릭
3. Microsoft 계정 로그인 또는 신규 생성
4. 학교 이메일 주소 입력 → 인증 메일 수신 → 클릭
5. 국가/학교/휴대폰 번호 등 입력 후 이용 약관 동의
6. 가입 완료 → 즉시 $100 크레딧 및 무료 서비스 사용 가능
7. 매년 재학생 신분( 학교 이메일**) 갱신 시 무료 혜택 재지원
<br><br><br>



## Microsoft 365
12개월 무료 체험 + 그 후 50% 할인된 월 $4.99에 Microsoft 365 Personal 이용 가능
🔹포함 항목: Word, Excel, PowerPoint, OneNote, Outlook 등 동기화 가능한 1 TB OneDrive 클라우드 저장소, Microsoft Defender 보안 기능, Copilot for AI 도구 포함
🔹대상: 대학(College) 또는 대학교 학생 (교직원 포함), 학교 발급 이메일 주소(예:john@university.edu)를 통해 확인<br>

🧭 **신청 절차**
1. [Microsoft 365 무료 학기 혜택 페이지] 방문 https://www.microsoft.com/ko-kr/education/products/office
2. 학교 이메일 입력 후 인증
3. 계정 생성/로그인, 결제 수단 등록
4. 12개월 무료 이용 시작 → 이후 할인 적용된 구독으로 전환
<br><br><br>



## Github Copilot Pro
**GitHub Education에 참여할 자격이 있나요?**
다음과 같은 경우 GitHub Education에 참여할 자격이 있습니다.
🔹고등학교, 전문 대학, 종합 대학교 또는 홈스쿨과 같은 학위나 졸업장 교부 프로그램에 등록되어 있습니다.
🔹학교에서 발급한 메일 주소 또는 학생 신분을 증명할 수 있는 문서를 보유하고 있어야 합니다.
🔹GitHub 개인 계정을 보유하고 있어야 합니다.
🔹만 13세 이상이어야 합니다.

**학생임을 증명하는 데 어떤 문서를 사용할 수 있나요?**
🔹현재 재학 중임을 증명할 수 있는 학교 ID 사진
🔹수업 시간표
🔹성적 증명서
🔹소속 증명서 또는 재학 증명서<br>


🧭 **신청 절차**

1. [Education benefits](https://github.com/settings/education/benefits) 으로 이동합니다.
2. "GitHub Education"에서 Start an application을 클릭합니다.
3. 양식을 완료한 다음, Submit application을 클릭합니다.

애플리케이션이 승인되면 [GitHub Education ](https://github.com/education) 포털로 이동하여 전문 개발자 도구에 무료 액세스 등의 GitHub Education 혜택을 이용할 수 있습니다.
